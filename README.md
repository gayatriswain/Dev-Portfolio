## 🚀 Features

- 🎨 **Modern UI/UX:** Glassmorphism, smooth animations, and a beautiful dark theme.
- 📱 **Fully Responsive:** Looks perfect on mobile, tablet, and desktop devices.
- 🛠️ **Admin Dashboard:** Edit your portfolio content dynamically without touching the code!
- 🕹️ **Interactive Elements:** 3D animated models, cursor-following characters, and hover effects.
- ⚡ **High Performance:** Built with Next.js for blazing fast load times and SEO optimization.

## 🛠️ Tech Stack

- **Frontend:** React, Next.js
- **Styling:** Tailwind CSS, Framer Motion
- **3D Graphics:** React Three Fiber, Drei
- **Icons:** Lucide React

## 🎯 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/gayatriswain/Dev-Portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```
4. Open `http://localhost:8080` in your browser.

## ⚙️ Admin Dashboard

You can edit all the contents of the portfolio (Projects, Skills, About section, Contact details) directly through the built-in Admin UI.

1. Navigate to `http://localhost:8080/admin`
2. Update the text, upload images, or change stats.
3. Save changes, and your portfolio updates instantly!
